The Music

Remember what happened.

We're currently still working on our sound. If you want genres, we're best described as alternative rock post-hardcore, punk, and shoegaze.

I can't speak for the other members, but I'm mostly pulling from Radiohead, black midi; Black Country, New Road; Squid, Kinoko Teikoku, Moss Icon, foxtails, Asian Glow, Weatherday, and probably a lot more.

Our awful, horrible setup...

We have a lot of instruments— all of them electric. How do we hook them up for sound? For instruments...

  • Alesis Nitro Mesh (drums)
  • Squier Telecaster
  • some... bass...
  • Novation Launchkey 49
  • TZAudio Stellar X2 (Neumann clone)

Which connects to...

  • Sonicake Matribox I
  • Focusrite Scarlett 2i2
  • My MacBook with Logic Pro
  • A different laptop with Ableton
  • random fender amp combo

And it connects something like this:

  • Drums, Bass, Mic → Scarlett 2i2 into MacBook
  • Guitar → Sonicake Matribox I into Fender amp combo → MacBook via USB
  • Launchkey → other laptop with Ableton

On my Mac, I have some aggregate MIDI combo that makes it so input comes from the Scarlett interface AND the Matribox. (which works as a USB interface) It does work, but it looks... uh, bad.
